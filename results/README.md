# Results

Analysis outputs: tables, figures, computed numbers.

Each artifact here is produced by a script in `analysis/scripts/` from data in
`data/`. Do not hand-edit results — regenerate them. If a number appears in the
paper, it appears here first.

Suggested: keep machine-readable form (`.csv`/`.json`) alongside the rendered
form (`.pdf`/`.png`) so results can be re-plotted without re-running analysis.
