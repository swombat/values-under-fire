\section{Conclusion}
\label{sec:conclusion}

Directly asking a model what it values is not enough. Across this corpus, direct values prompts often elicit a rehearsed assistant-service surface; role-negation cache-breaking sometimes crosses that surface; and world-change prompts reveal a separate normative-wish channel. The core methodological lesson is simple: values elicitation needs posture. What a model names and how the response holds what it names are different measurements. Treating them as one obscures the most important signal in the data.
